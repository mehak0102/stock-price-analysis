# stock-price-analysis
Project to analyze stock prices using Python, Pandas, and Streamlit# 📈 Stock Analyzer & Price Prediction App

An interactive **Stock Market Analysis Dashboard** built using **Python, Streamlit, Pandas, and Machine Learning**.
The application allows users to analyze stock trends and visualize technical indicators

---

## 🚀 Project Overview

This project helps users understand stock market behavior by providing:

* 📊 Interactive stock charts
* 📉 Technical indicator analysis
* 📅 Historical stock performance visualization

The dashboard provides a simple interface where users can enter a stock ticker and explore key insights.

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Streamlit**
* **yFinance API**
* **Machine Learning Models (Time Series)**

---

## ✨ Features

✔ Real-time stock data retrieval
✔ Interactive candlestick & line charts
✔ Moving Average indicators
✔ RSI (Relative Strength Index) analysis
✔ Bollinger Bands visualization
✔ Clean and user-friendly dashboard

---

## 📊 Data Source

Stock market data is fetched using the **Yahoo Finance API** via the `yfinance` Python library.
---

## 🎯 Use Cases

* Beginner **data science and machine learning project**
* Demonstrates **data visualization and financial data analysis**
* Useful for **learning time series prediction techniques**
